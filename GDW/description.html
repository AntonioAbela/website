HTML Terms

HTML
Stands for Hypertext Markup Language, it is a language that tells a browser how to display text and images
HTML Element
The content of HTML
HTML Tag
What holds the HTML elements
Angle Bracket
Used to close code
HTML Attribute
Words inside a tag that control the elements
HTML Comment
Text that is seen in the code but not on the webpage
HTML Object element
A container for an external source
HTML Special Character
Text used to represent characters that would be considered code in HTML
